# Laputa App

## Build & Dev

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ npm run dev

# build for production with minification
$ npm run build

# lint code
$ npm run lint

# build + docker build
$ npm run build:docker

# 在本地 build docker 镜像
$ npm run docker:build

# 在本机测试运行 build 好的镜像
$ npm run docker:run
```
